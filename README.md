## 📖 About the Project
This repository contains the source code for my personal portfolio website. It was built from scratch to demonstrate not just my professional experience and projects, but also my ability to create responsive, interactive web experiences without relying on heavy frontend frameworks. 

The site features custom CSS art, interactive machine learning flow simulations, and a fully responsive grid system optimized for all devices.

## ✨ Key Features
* **Interactive CSS Animations:** Features custom-built animated scenes including a simulated Neural Network flow, a Deep Learning orbit visualizer, and a REST API network diagram.
* **Fully Responsive Design:** Fluid typography and adaptive grid layouts ensure a seamless experience on ultra-wide monitors, tablets, and mobile devices.
* **Dynamic UI Elements:** Includes a custom glowing cursor effect, typing text animations, and intersection observers for smooth scroll-reveal effects.
* **Project Showcase:** Dedicated sections for deployed Machine Learning models (Classification & Regression) and full-stack enterprise applications.
* **Contact Integration:** Direct email routing via native mailto forms and quick links to GitHub and LinkedIn.

## 🛠️ Tech Stack
* **HTML5:** Semantic markup and structure.
* **CSS3:** Flexbox, CSS Grid, complex keyframe animations, CSS variables, and glassmorphism UI design.
* **Vanilla JavaScript:** DOM manipulation, scroll-spying navigation, and dynamic cursor tracking.
* **Font Awesome:** For scalable vector icons.

## 💻 Running Locally
To view or modify this project on your local machine:

1. Clone the repository:
   ```bash
   git clone [https://github.com/kamilperwez/Kamil-Perwez-Portfolio.git](https://github.com/kamilperwez/Kamil-Perwez-Portfolio.git)
   ```
2. Navigate to the project directory:
   ```bash
   cd Kamil-Perwez-Portfolio
   ```
3. Open `index.html` in your preferred web browser. No build steps or package installations are required!
